Gestão de Clientes

Aplicação feita com Django 2.2 e Python 3 para fins de aprendizado

Utilização:
-Abrir pasta do projeto com PyCharm
-Navegar via terminal até a pasta gestao_clientes
-Rodar o comando 'python manage.py migrate'
-Rodar o comando 'python manage.py runserver' para iniciar o servidor
-Criar superuser com o comando 'python manage.py createsuperuser'
-Acessar via web 'localhost:8000'
-Neste momento ira se deparar com a pagina home, clique no botão "Entrar"
-Autenticar-se com as credenciais de superusuario
-Sera direcionado para pagina de clientes, onde pode cadastrar, alterar, e deletar os mesmos.
